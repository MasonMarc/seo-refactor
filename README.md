### Description

This project is a website for a Search Engine Optimization and Social Media Management company that has semantic elements and accessibility features. These include semantic tags in the html as well as alt descriptions for every image, meta data to make the sight more visible on search engines, making a concise title for the page, and making sure the images are a size that is easy to load while maintaining fidelity.
